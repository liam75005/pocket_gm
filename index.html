Pourquoi une clé API ?

Pocket GM utilise Claude comme Maître du Jeu. Votre clé API permet à l'app de communiquer avec Claude depuis votre navigateur, sans serveur intermédiaire.

🔒 Sécurité : Votre clé reste dans votre navigateur (localStorage). Elle n'est envoyée qu'à api.anthropic.com, jamais ailleurs.

💰 Coût : ~$1-3 par aventure complète avec Sonnet 4.6, ~$0.30-1 avec Haiku 4.5. Le cache de prompt réduit fortement le coût des appels successifs.

📋 Comment obtenir une clé :

  1. Créez un compte sur console.anthropic.com
  2. Ajoutez du crédit (5-10$ suffisent pour des dizaines d'heures de jeu)
  3. Allez dans API Keys → Create Key
  4. Copiez la clé (commence par sk-ant-) et collez-la ci-dessous
